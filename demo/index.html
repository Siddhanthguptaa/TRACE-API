What is TRACE?

As AI agents autonomously execute micro-transactions, a massive security hole has opened. Malicious actors can spin up 10,000 cryptographic identities in seconds to farm marketplaces and execute coordinated collusion attacks. TRACE is the firewall.

Real-time Scoring

Sub-50ms trust scores using Bayesian LCB, CUSUM change-point detection, and Personalized PageRank. No LLM calls, deterministic latency.

Sybil & Collusion Detection

Mathematically detects coordinated edge injection, Sybil clusters, and disguise-collusion rings. 14× more resilient than EigenTrust against N=5000 agent attacks.

Stateful Trust Graph

Global trust graph maintained server-side. Events update CUSUM state, trust edges, and PageRank asynchronously. Every marketplace strengthens the shared graph.

Drop-in x402 Middleware

One-line integration for Python/TypeScript. Gates x402 USDC payments before processing. Returns ROUTE/HOLD/QUARANTINE decisions with explanations.

Enterprise Audit Trail

Full billing history, event deduplication, and immutable trust records. PostgreSQL persistence with Alembic migrations. SOC2-ready architecture.

Prometheus Metrics

Built-in observability: request latency, score distributions, worker health, trust graph stats, and billing metrics. Grafana dashboards included.

One-Line Integration

Drop the TRACE middleware into any FastAPI, Flask, Express, or Hono app. It gates x402 payments before any funds move.

Python (FastAPI) 
from sdk.x402_middleware import TRACEMiddleware
from fastapi import FastAPI, Request

app = FastAPI()
trace = TRACEMiddleware(api_key="sk_trace_...")

@app.post("/agent/summarize")
async def handle_summarize(request: Request):
    agent_wallet = request.headers.get("X-Payment-Sender")

    # Raises HTTPException(402) if untrusted
    score = await trace.check(agent_wallet, "summarize", 0.05)

    # Agent passed trust check — proceed
    return {"result": "..."}

import { traceGate } from "./x402_middleware";

app.use("/agent", traceGate({ apiKey: "sk_trace_..." }));

  res.json({ result: "..." });
});
